# siddharthverma23.github.io

Personal portfolio website — built with vanilla HTML/CSS/JS, hosted on GitHub Pages.

## Local Development

Open `index.html` in a browser, or run a local server:

```bash
python3 -m http.server 8080
```

## Deployment

Pushing to `main` automatically deploys via GitHub Pages.

**Live:** [siddharthverma23.github.io](https://siddharthverma23.github.io)
