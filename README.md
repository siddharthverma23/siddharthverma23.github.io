# Siddharth Verma — Portfolio Website

A modern, responsive portfolio website built for GitHub Pages.

## 🚀 Quick Deploy to GitHub Pages

1. Create a new repository on GitHub named `siddharthverma23.github.io`
2. Push this code:

```bash
cd Portfolio_Website
git init
git add .
git commit -m "Initial portfolio website"
git branch -M main
git remote add origin https://github.com/siddharthverma23/siddharthverma23.github.io.git
git push -u origin main
```

3. Go to **Settings → Pages** in your repo and ensure the source is set to `main` branch, root `/`
4. Your site will be live at: **https://siddharthverma23.github.io**

## 📁 Structure
